Shorten URLs with ltlurl!

Pass in a url - Receive a shortened url that will redirect to the original url

Example Input:

https://ltlurl.herokuapp.com/new/https://www.google.com

Example Output:

{"original_url":"https://www.google.com","short_url":"https://ltlurl.herokuapp.com/rJyprLfi"}

Usage:

https://ltlurl.herokuapp.com/rJyprLfi

Will redirect to:

https://www.google.com
